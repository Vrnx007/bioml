# Bip Mp

B2B marketplace UI. Compound names, CAS, and 2D structure images may reference [PubChem](https://pubchem.ncbi.nlm.nih.gov/) (NCBI) via PUG REST; PubChem does not sell chemicals.

## Local

```bash
npm install
npm run dev
```

Open `/en/catalog`. Auth and seeded DB: see `docs/DEMO_ACCESS.md` and `supabase/seed.sql`.
